# Vue3 流程图编辑器

一个基于Vue3的可靠性软件元器件库流程图编辑器，支持拖拽、连接和参数配置。

## 功能特性

- 📋 流程图列表管理
- ✏️ 可视化流程图编辑器
- 🎨 15种电力系统组件
- ⚙️ 组件参数配置
- 🔗 智能连接线
- 💾 保存/加载功能
- 📱 响应式布局

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 使用说明

1. 进入流程图列表页面
2. 点击"新建流程图"或现有流程图进入编辑器
3. 从左侧组件面板拖拽组件到画布
4. 拖动组件到相邻位置可以连接
5. 点击组件打开配置面板设置参数
6. 点击保存按钮保存流程图
